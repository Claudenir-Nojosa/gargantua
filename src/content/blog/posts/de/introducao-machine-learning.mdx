# Introdução ao Machine Learning

Machine Learning (Aprendizado de Máquina) é uma subárea da Inteligência Artificial que permite que sistemas aprendam com dados, identifiquem padrões e tomem decisões com intervenção humana mínima.

## O que é Machine Learning?

Machine Learning é a ciência de fazer computadores aprenderem sem serem explicitamente programados.

### Tipos de Machine Learning

1. **Aprendizado Supervisionado**
2. **Aprendizado Não Supervisionado**
3. **Aprendizado por Reforço**

## Primeiros Passos

Para começar com Machine Learning, você precisa:

- Conhecimento básico de Python
- Familiaridade com matemática (álgebra linear, cálculo, estatística)
- Pacotes como NumPy, Pandas, Scikit-learn

## Exemplo de Código em Python

```python
# Importando bibliotecas
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

# Criando dados de exemplo
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

# Dividindo em treino e teste
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Criando e treinando o modelo
model = LinearRegression()
model.fit(X_train, y_train)

# Fazendo previsões
predictions = model.predict(X_test)
print(f"Previsões: {predictions}")
